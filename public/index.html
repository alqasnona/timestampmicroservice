Timestamp Microservice

API Project: Timestamp Microservice

User Stories:

  1. I can pass a string as a parameter, and it will check if it contains either a Unix timestamp or a valid date string (e.g. "2015-12-25")
  2. If it is valid, it returns both the Unix timestamp and the UTC string format
  3. If it is invalid, it returns an error object
  4. When I visit without a date parameter, it returns the current time in both formats

Example Usage:

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000
  • [project url]/api (current date)

Example Output:

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}
